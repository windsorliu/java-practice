此程式可印出三種圖形

### 普通三角形
```
    *  
   ***  
  ***** 
```

### 菱形
```
    *
   ***
  *****
   ***
    *
```

### 空格三角形
```
     *
    * *
   * * *
  * * * *
```

## 思路
### 普通三角形

line &rarr; 總層數  0
i &rarr; 該層層號 
```
line = 4

第1層  000* 
第2層  00***  
第3層  0*****  
第4層  *******  
```
空格數 = 總層數 - 該層層數 = line - i  
星星數 = 該層層數 * 2 - 1 = i * 2 - 1  

### 菱形

```
line = 3 

1  00*    
2  0***
3  *****
4  0***
5  00*
```
菱形上半部與普通三角形邏輯相同<br>
下半部會比上半部少一層，並且空格數與星星數增加邏輯與普通三角形相反(倒三角形)

### 空格三角形
```
1     *
2    * *
3   * * *
4  * * * *
```
空格部分邏輯相同於普通三角形<br>
在印出 * 時，只有奇數會印出 * ，偶數則印出空格
